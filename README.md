# 📰 Fake News Detection App

This project is a Machine Learning web app that detects whether news is Fake or Real using Streamlit.

## 🚀 Features
- Detect Fake and Real News
- Machine Learning model (Logistic Regression)
- User input prediction
- Streamlit web interface

## 🧠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## 📂 Project Files

app.py  
Fake.csv  
True.csv  
requirements.txt  
README.md  

## ▶️ How to Run

Install libraries:

pip install -r requirements.txt

Run app:

streamlit run app.py
